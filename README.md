## Technical stack (MERN)
1. Client
  1. Languages: React
  2. Frameworks: Material UI, Styled Components
  3. Host: AWS Cloud

2. Server
  1. Runtime Environment: Node.js
  2. Frameworks: Express.js
  3. Database: MongoDB
  4. Payment Processing Service Provider: Stripe
